## Install with Chef-DK

Install via `chef gem` subcommand.

```
$ chef gem install knife-zero
```

## Install from Rubygems

Add this line to your application's Gemfile:

```ruby
gem 'knife-zero'
```

And then execute:

```
$ bundle
```

Or install it yourself as:

```
$ gem install knife-zero
```
